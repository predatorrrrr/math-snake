# 有向數貪食蛇 Math Snake

一個幫助學生學習「有向數（正負數）」的貪食蛇遊戲，介面以 SVG 與 CSS 打造擬真效果（草地紋理、木質圍欄、立體蛇身、光影果實與互動數軸）。

## 玩法

1. 每一關會出現一個「目標總和」（可能是正數或負數）。
2. 操控小蛇吃下場上的果實：
   - 🍎 **蘋果（正數）**：總和「加上」該數，例如吃到 `+3` 就是 `+ (+3)`。
   - 🫐 **藍莓（負數）**：總和「加上」負數，例如吃到 `−5` 就是 `+ (−5)`，即往回減 5。
3. 讓「目前總和」剛好等於「目標總和」即可過關；下方數軸會即時顯示小蛇目前所在的數值位置與目標旗幟。
4. 撞到圍欄或咬到自己身體則遊戲結束。每過一關速度會加快，數字範圍也會變大。

## 操作

| 按鍵 | 功能 |
| --- | --- |
| 方向鍵 / WASD | 移動 |
| 空白鍵 | 暫停 / 繼續 |
| 觸控裝置 | 畫面下方的方向按鈕 |

## 執行方式

純靜態網頁，無任何相依套件。直接用瀏覽器開啟 `index.html`，或啟動任一靜態伺服器：

```bash
python3 -m http.server 8000
# 開啟 http://localhost:8000
```

## 專案結構

```
index.html      頁面結構與 SVG 定義（漸層、紋理、濾鏡）

js/game.js      遊戲邏輯、SVG 繪製、數軸與音效
```
