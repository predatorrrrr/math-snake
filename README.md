# 有向數貪食蛇 Math Snake

一個幫助學生學習「有向數（正負數）」的連續答題貪食蛇遊戲，介面以 SVG 與 CSS 打造擬真效果（草地紋理、木質圍欄、立體蛇身、光影果實與神奇道具）。

## 玩法

1. 在主選單選擇難度：
   - **簡單**：兩個有向數加減（±9）
   - **中等**：三個有向數加減（±12）
   - **困難**：有向數乘法、除法與混合運算
2. 畫面上方會出現一條算式（例如 `(−7) + (+8) − (−1) = ?`），計算出答案後，操控小蛇吃掉標示**正確答案**的果實：
   - 🍎 紅蘋果＝正數　🫐 藍莓＝負數　🍋 檸檬＝零
3. 答對 **+20 分**（連擊有額外加分），馬上出下一題，遊戲連續進行不停歇。
4. 吃錯答案 **−10 分**，而且小蛇會**變長兩節**作為懲罰，越來越難操控！
5. 撞到圍欄（沒有穿牆術時）或咬到自己身體，遊戲結束。

## 神奇道具

場上會不定時出現道具，吃到立即生效：

| 道具 | 效果 |
| --- | --- |
| 🌀 穿牆術 | 10 秒內可以穿過圍欄，從另一邊出來（圍欄會泛紫光） |
| ⏰ 慢動作 | 8 秒內小蛇減速，方便思考 |
| 🍄 縮小菇 | 立即縮短三節蛇身 |

## 操作

| 按鍵 | 功能 |
| --- | --- |
| 方向鍵 / WASD | 移動 |
| 空白鍵 | 暫停 / 繼續 |
| 觸控裝置 | 畫面下方的方向按鈕 |

## 執行方式

純靜態網頁，無任何相依套件。直接用瀏覽器開啟 `index.html`，或啟動任一靜態伺服器：

```bash
python3 -m http.server 8000
# 開啟 http://localhost:8000
```

## 專案結構

```

js/game.js      遊戲邏輯、出題系統、道具系統、SVG 繪製與音效
```
